# Deployment Steps

This document describes how to deploy the AWS 3-Tier architecture.

1. Create VPC with public and private subnets
2. Configure Internet Gateway and route tables
3. Launch EC2 instances with Auto Scaling
4. Configure Application Load Balancers
5. Deploy RDS in private subnets
6. Apply security groups and IAM roles
7. Enable monitoring and logging
